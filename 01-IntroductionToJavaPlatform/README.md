# JDK vs JRE vs JVM

- JVM (Java Virtual Machine) runs Java bytecode.
- JRE = JVM + Libraries + Other Components 
- JDK = JRE + Compilers + Debuggers



##### In the java bin folder, open a Command Shell and run

    javac HelloWorld.java
